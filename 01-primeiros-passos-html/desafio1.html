Estrutura básica do HTML

HTML é uma linguagem de marcação composta por tags que envolvem o conteúdo e o estruturam de forma hierárquica. Cada página HTML começa com uma tag DOCTYPE html, indicando a versão do HTML que está sendo usada. O elemento html engloba todo o conteúdo da página, enquanto head contém informações como o título da página, links para folhas de estilo e meta informações.

Resumo:

  1. Tag html: Tag usada para delimitar que todo o contéudo dentra da mesma possui a um arquivo HTML. (Obrigação de fechamento).
  2. Tag head:Tag para delimitar a configurações do arquivo HTML, as informações presentes nesta tag são pré carregadas antes da exibição do contéudo.
  3. Tag body:Tag usada para delimitar todo o contéudo visivel para o cliente no nevegador.

Falando sobre tags

Tags HTML em sua maioria possuem abertura e fechamento, e todo o contéudo entre uma e outra recebe a propriedade da tag em questão.

Algumas tags não possuem fechamento, o fechamento está embutido dentro da propria tag.

Outra dica é a sobreposição de tags, é permitido aninhar tags, exemp´lo transformar um texto em negrito e itálico.


Atributo das tags

Atributos são propriedades que uma determianda tag pode possuir.

Importante salientar que existem atributos padrões que são permitidas em qualquer tag HTML, mas existem alguns atributos únicos para tag sem especifico.

Atributos mais comuns

  1. id: Fornece um identificador exclusivo para um único elemento na página.
  2. class: Fornece um nome de classe que pode ser aplicado a múltiplos elementos.
  3. style: Permite aplicar regras de CSS diretamente ao elemento HTML.

Atributos especifícos

  1. type: Indica o tipo de dado que está sendo transmitido ou o comportamento do controle de formulário.
  2. src: Especifica a URL (caminho) do arquivo externo que o elemento deve carregar.

Textos

Tags para estruturas básicas de conteúdo

  • Tag h1 ao h6

    Define títulos e subtítulos de uma seção. h1 é o mais importante (nível 1) e deve ser usado apenas uma vez por página para o título principal. h6 é o menos importante (nível 6).

  • Tag p

    Define um parágrafo de texto. É a tag mais comum para blocos de texto.

  • Tag div

    Define uma divisão ou seção de um documento. É uma tag genérica usada para agrupar conteúdo em bloco e aplicar estilos (CSS) ou manipulá-lo (JavaScript).

  • Tag br

    Insere uma quebra de linha única.

  • Tag hr

    Define uma linha horizontal temática, usada para separar visualmente seções de conteúdo.

  • Tag pre

    Define um texto pré-formatado. O texto dentro dela é exibido em uma fonte de largura fixa e mantém os espaços em branco e quebras de linha exatamente como foram escritos no código.


Listas Ordenadas e Não Ordenadas

Lista Não Ordenada

É usada quando a ordem dos itens não é importante. Cada item da lista é marcado por um marcador de ponto (bullet point) padrão.

Tag: < ul >

Lista Ordenada

É usada quando a ordem dos itens é importante ou sequencial (por exemplo, um passo a passo, um ranking ou um índice). Cada item da lista é marcado por um número ou letra por padrão.

Tag: < ol >

Links

A Tag < a > (Âncora)

O elemento < a > é usado para criar um hiperlink para outras páginas, arquivos, endereços de e-mail, locais dentro da mesma página ou qualquer outro recurso que uma URL possa endereçar.

Sintaxe Básica

< a href="URL_DE_DESTINO" >Texto do Link< /a >
